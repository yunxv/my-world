## SS 的小世界

一个温馨的游戏心情记录工具。

### 功能特点

- 📸 上传游戏照片并记录心情
- 📅 按时间倒序、按年/月分组展示
- ✏️ 支持编辑、删除记录
- 🔍 支持文字/分类/日期搜索
- 🐱 每条记录会得到随机动物叫声回复
- 💾 数据保存在浏览器 LocalStorage（刷新后不丢失）

### 打开方式

直接在浏览器中打开 `index.html` 即可使用。

### 使用说明

1. 点击"+ 记录美好时刻"按钮开始记录
2. 选择游戏分类（时空/宇宙）
3. 上传照片并写下心情
4. 保存后会得到可爱的动物叫声回复
5. 使用搜索功能快速找到过往记录

### 技术说明

- 纯前端实现，无需服务器
- 使用 LocalStorage 本地存储数据
- 响应式设计，支持手机端使用

### 配色方案

- 主色调：浅紫色（薰衣草紫）
- 辅色调：清新绿（淡绿色）
- 风格：水彩手绘

### 数据来源

所有数据保存在浏览器本地，清除浏览器数据会丢失记录。
